🛡️ Cybersecurity: AI-Powered Threats Spike

Las amenazas cibernéticas impulsadas por IA alcanzaron niveles récord en abril 2026. IC security enfrenta desafíos de quantum, AI y automotive attack surfaces.

Incidentes destacados

  • Claude AI descubre RCE: CVE-2026-34197 en Apache ActiveMQ, vulnerabilidad de 13 años
  • ValleyRAT + GitHub: weaponizado contra devs vía repositorios poisoned + gaming PCs
  • GitHub Copilot CamoLeak (CVSS 9.6): exfiltración masiva de API keys y source code
  • Booking.com breach: datos personales de clientes expuestos
  • Basic-Fit: ~1M clientes afectados, datos bancarios robados
  • Phishing AI: device code phishing targeting cuentas Microsoft enterprise
  • Email gubernamentales: miles de cuentas con contraseñas en plaintext halladas online

Defensa

  • Post-quantum cryptography: hardware trust anchors, secure boot, protected key storage
  • Zero Trust Architecture: adoptada como estándar en empresas tech
  • Supply chain security: multi-vendor complexity genera exposición
